# Luau Type Utils

A collection of various type utilities for the New Luau Type Solver.

## Installation

### GitHub

Download the `.rbxm` file from the [Latest Release](https://github.com/Nowoshire/LuauTypeUtils/releases/latest) and place it in your desired location.

### Wally

```toml
TypeUtils = "nowoshire/luautypeutils@^0.0.0"
```

### Rojo (Build from Source)

```bash
git clone https://github.com/Nowoshire/LuauTypeUtils.git
cd LuauTypeUtils
rojo build --output "../TypeUtils.rbxm"
```
